# Chyngyz's Dungeon Game

## Описание проекта

Chyngyz's Dungeon Game — учебный 2D экшен‑проект, разработанный на игровом движке Godot. Игрок управляет рыцарем, который проходит подземелье и сражается с монстрами. Проект выполнен в рамках учебной работы и предназначен для демонстрации базовых навыков разработки игр.

На текущий момент в игре реализованы три типа противников, система боя, меню, пауза и несколько игровых сцен.

## Техническая информация

* Игровой движок: Godot Engine 4.x
* Язык программирования: GDScript
* Платформа: PC (Windows)
* Разрешение экрана: 1280x720
* Целевая частота кадров: 10 FPS

---

## Сохранения

* Файлы сохранений хранятся в каталоге user:// (стандарт Godot)

---

## Запуск проекта

1. Установить Godot Engine версии 4.x
2. Запустить Godot
3. Выбрать пункт Import Project
4. Указать папку с проектом
5. Запустить сцену menu.tscn
